# CryptoDominance-Tracker
Durante minhas férias, acabei ficando curioso e um pouco entediado, então resolvi explorar APIs públicas de criptomoedas e testar algo prático. Este projeto coleta dados atualizados do Top 10 criptoativos, calcula dominância de mercado e gera automaticamente uma planilha integrada ao Power BI para análise visual em tempo real.
